## How to Embed This on Your Blog
Copy the exact internal iframe code and paste it into any WordPress Custom HTML block.

```html
<iframe src="https://policymaturity-tools.github.io/lic-premium-calculator-india/" width="100%" height="600" style="border:none; border-radius: 12px; box-shadow: 0 4px 15px rgba(0,0,0,0.1);"></iframe>
```

When users need the precise, down-to-the-rupee calculation for riders (like Critical Illness or Accident Benefit), the widget directs them to the exact calculators on policymaturity.in.
